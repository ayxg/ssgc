Detailed Description

Primary Expression Shift-Reduction Parser.

Cursor begin and end must be the start and end of the expression. Note this parser does not take the entire source as an argument. Only pass the current scope to be parser. The end of TkCursor may or may not be end of source. This parser does not advance the cursor or return a new start point.

◆ ChooseAction()

cxx::BoolError caoco::parser::ExprParser::ChooseAction ( TkCursor & c)
inlineprivate

[Choose Action] Decides which action to take. Main loop of the closure resolver. Action will be skip , store, or check.

  • On first occurence of an operator- a store will be performed. All following operators will be checks.
  • A skip is performed when an operand is encountered. Multi-token operands trigger repeated skips.
  • A check is performed when an operator is encountered.
  • At the end, always do a check.
