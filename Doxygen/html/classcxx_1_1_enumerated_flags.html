Detailed Description

template<mta::req::iEnumeration EnumT, mta::req::iIntegral UnderlyingT = std::underlying_type_t<EnumT>>
class cxx::EnumeratedFlags< EnumT, UnderlyingT >

Represents a set of flags that can be combined using bitwise OR.

Template Parameters
EnumTAn enumeration type which will be used as flags.
UnderlyingTDefaults to the underlying type of EnumT.

Flags may be appended or removed. Retrieve the flags using Get().
