\doxysubsection{Detailed Description}
\doxylink{classcaoco_1_1_builder}{Builder} for the C\& Official Compiler. 


\begin{DoxyItemize}
\item Handles the build pre-\/processing includes the lexing, parsing, macro expansion, constevaluator and code generation phases.
\item Responsible for managing the source files and temporary files.
\item Provides the interface for the user to build an executable of library.
\item Provides an interface to set the build options. 
\end{DoxyItemize}
