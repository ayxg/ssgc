\hypertarget{caoco_docs_minitest_autotoc_md14}{}\doxysection{\texorpdfstring{Lightweight single header unit testing framework for personal and small C++ 20 and above projects.}{Lightweight single header unit testing framework for personal and small C++ 20 and above projects.}}\label{caoco_docs_minitest_autotoc_md14}
Mini-\/\+Test has a familiar interface emulating the Google Testing Framework. When it\textquotesingle{}s time to move on to a more robust testing framework such as Google Test, only minimal refactoring is required. Mini-\/\+Test has no dependencies except the C++ standard library, and nothing to build -\/ simply copy \doxylink{minitest_8h}{minitest.\+h} into your project and include it.

Requires\+:
\begin{DoxyItemize}
\item Compiler flag minimum /std\+:c++20
\item P0315\+R4 \+: Lambdas in unevaluated contexts \href{https://www.open-std.org/jtc1/sc22/wg21/docs/papers/2017/p0315r4.pdf}{\texttt{ https\+://www.\+open-\/std.\+org/jtc1/sc22/wg21/docs/papers/2017/p0315r4.\+pdf}}
\end{DoxyItemize}

This is a single header file unit testing framework. Tests are defined as lambdas in a template parameter, which is executed upon initialization of static variables. As a result the order of tests is not guaranteed but the usual order of static initialization is followed. New\+: now supports fixtures and inline tests which can be run at a later time and defined inside methods.

Configuration Macros These macros are used to set up how the minitest library should behave\+:


\begin{DoxyItemize}
\item MINITEST\+\_\+\+CONFIG\+\_\+\+RECORD\+\_\+\+ALL\+: If true, all test results are recorded and can be viewed using MINITESTS\+\_\+\+RECORDED macro.
\item MINITEST\+\_\+\+CONFIG\+\_\+\+NO\+\_\+\+CONSOLE\+\_\+\+PRINT\+: If true, no output is printed to the console during the test run.
\item MINITEST\+\_\+\+CONFIG\+\_\+\+CUSTOM\+\_\+\+SEPARATOR\+: If defined, a custom separator is used between test sections which should be the macro def. Default is a dashed line. Definition must be a string literal with a newline.
\end{DoxyItemize}
