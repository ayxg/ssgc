\doxysubsection{Detailed Description}
Empty struct which defines traits of C\& compiler errors. 


\begin{DoxyItemize}
\item Encapsulates all the error type definitions.
\item Provides a uniform interface to access error detail printing methods of each error type.
\end{DoxyItemize}

Use the method Err\+Detail$<$\+T$>$ where T is the error type. Each Err\+Detail method may have unique args but always returns a std\+::string.
